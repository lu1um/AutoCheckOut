AutoCheckOut by lu1um

1. AutoCheckOut 폴더안에 필요파일인
  URL.txt
  SURVEY.txt
   를 추가해주세요.

2. 설명서와 같은 폴더에 있는 AutoCheckOut.sh 실행
  같은 폴더내의 정상실행화면.jpg 처럼 뜬다면 성공
3. 실행시 라이브러리 설치가 자동으로 되지만, 만약 안된다면
  pip install pyqt5
  pip install selenium
   두 가지 모두 설치 후 실행
4. 실행 후 원하는 모드 선택 후에 ID, PW, 설문조사를 작성하고 activate 누르기
 - 출석 선택시
    출석, 퇴실 시간에 맞춰 실행됨
 - 로그인만 선택시
    바로 로그인 되고, 출석이나 퇴실, 설문조사는 실행하지 않음

 - slow mode 선택시
    페이지가 넘어가고 2초정도 기다립니다. 컴퓨터가 안좋거나 뭐하는지 보고싶을때만 선택

* 주의사항 *
- 처음 실행하면 새로운 라이브러리 설치로 인해 켜지는데 시간이 걸립니다. 
  3분정도가 지나도 켜지지 않는다면 다시 AutoCheckOut.sh 파일을 실행해주세요.
- 프로그램이 실행되면서 새로 켜진 크롬은 끄지 말아주세요.
- 크롬이 없어졌거나, 잘 못 선택했거나, 뭔가 문제가 있다면 Reboot Chrome을 눌러주세요.
- 프로그램을 새로 실행할 때에는 먼저 실행한 창을 끄고나서 실행해주세요.
