1. AutoCheckOut 폴더안에 
  URL.txt
  SURVEY.txt
   를 추가한다. (직접 만드는 것이 아니고 다른데서 다운받기)

2. 설명서와 같은 폴더에 있는 AutoCheckOut.sh 실행
3. 실행시 라이브러리 설치가 자동으로 되지만, 만약 안된다면
  pip install pyqt5
  pip install selenium
   두 가지 모두 설치 후 실행
4. 실행 후 원하는 모드 선택 후에 ID, PW, 설문조사를 작성하고 activate 누르기
 - 출석 선택시
    출석, 퇴실 시간에 맞춰 실행됨
 - 로그인만 선택시
    바로 로그인 되고, 출석이나 퇴실, 설문조사는 실행하지 않음
